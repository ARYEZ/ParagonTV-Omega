# Chapter 1: Introduction & Overview

## What is Paragon TV?

Paragon TV transforms your Kodi media center into a sophisticated personal television network. Instead of browsing through endless menus to find something to watch, Paragon TV creates custom TV channels from your media library - giving you the authentic channel-surfing experience you remember, but with content you actually own and love.

## Why Paragon TV?

**The Problem:**
Modern streaming services and media libraries require constant decision-making. You spend more time browsing than watching, suffering from "choice paralysis."

**The Solution:**
Paragon TV brings back the simplicity and comfort of traditional television:
- **Just turn it on** - Something's already playing
- **Surf through channels** - Like flipping through your favorite networks
- **Discover content** - Stumble upon shows you forgot you owned
- **Set it and forget it** - Automated scheduling handles everything

## Key Features at a Glance

- **Custom Channels** - TV shows, movies, music, and smart playlists
- **Smart Distribution** - Ensures fair episode rotation across your library
- **10-Phase Automation** - Schedule your TV day from morning to night
- **Speed Dial** - Instant access to favorite channels and shows
- **Multi-Room Sync** - Coordinate playback across multiple TVs
- **Weather Overlay** - Live weather display while watching
- **Episode Browser** - Jump to any episode without leaving your show
- **Favorite Notifications** - Get alerts when your favorite shows are about to start
- **Paragon Sentry** - Visual channel monitoring with real-time artwork

## Technical Requirements

- **Kodi Version:** 17.6 (Krypton)
- **Python:** 2.7 runtime
- **Storage:** Sufficient space for channel configurations (typically 50-100MB)
- **Network:** For multi-room sync and weather features (optional)
- **Operating System:** Windows, Linux (LibreELEC recommended), macOS, Android

# Chapter 2: Getting Started

## Installation

### Step 1: Download Paragon TV
1. Download the latest Paragon TV package (`script.paragontv-x.x.x.zip`)
2. Save it to a location you can access from Kodi

### Step 2: Install the Add-on
1. Open Kodi
2. Go to **Add-ons** from the main menu
3. Click the **Package Icon** (üì¶) at the top left
4. Select **Install from zip file**
5. Navigate to your downloaded `script.paragontv-x.x.x.zip`
6. Wait for the installation confirmation

### Step 3: Initial Configuration
1. After installation, Paragon TV will appear in your **Program Add-ons**
2. Launch it for the first time
3. You'll be guided through basic setup:
   - Library scan confirmation
   - Auto-start preferences
   - Basic channel creation

## First Launch

When you launch Paragon TV for the first time:

1. **Library Scan** - Paragon TV will scan your Kodi library to discover available content
2. **Channel Creation** - Basic channels will be auto-created based on your library
3. **EPG Generation** - The Electronic Program Guide will be populated
4. **Ready to Watch** - You'll land on Channel 1 with content already playing

## Basic Navigation

**Remote Control:**
- **Up/Down Arrows** - Change channels
- **OK/Select** - Open Program Guide (EPG)
- **Info** - Show currently playing information
- **Back** - Return to previous screen/Close overlay
- **Number Keys (0-9)** - Direct channel access or Speed Dial

**EPG (Program Guide):**
- **Left/Right** - Scroll through time
- **Up/Down** - Navigate between channels
- **OK** - Jump to selected program
- **Menu** - Access settings and configuration

# Chapter 3: Channel Types & Configuration

## Understanding Channel Types

Paragon TV supports four main channel types:

### Type 0: Smart Playlist Channels
**What it does:** Creates channels from Kodi Smart Playlists  
**Best for:** Custom collections (Recently Added, Highly Rated, Genre mixes)  
**Example:** "90s Action Movies" or "Watched Last Month"

**How to create:**
1. Create a Smart Playlist in Kodi first
2. In Paragon TV Config, select **Type 0: Playlist**
3. Browse to your playlist file
4. Set playback mode and limits
5. Save

### Type 3: TV Show Channels
**What it does:** Dedicates a channel to TV shows by genre or title  
**Best for:** Genre channels (Comedy, Drama, Sci-Fi) or single-show channels  
**Example:** "Comedy Central" showing all your sitcoms

**Special Feature - Smart Distribution:**
- Every show gets at least 1 episode
- Maximum 5% per show (5 episodes per 100 total)
- Weighted selection favors smaller shows
- Episode spacing prevents clustering

**How to create:**
1. Select **Type 3: TV Shows**
2. Choose genre or manually select shows
3. Smart Distribution is automatic
4. Configure shuffle and repeat options

### Type 4: Movie Channels
**What it does:** Creates movie-only channels by genre  
**Best for:** Movie marathons (Horror Night, Rom-Com channel)  
**Example:** "Action Theater" or "Friday Frights"

**How to create:**
1. Select **Type 4: Movies**
2. Pick a genre from your library
3. Set shuffle and repeat preferences
4. Configure bumpers and commercials (optional)

### Type 12: Music Channels
**What it does:** Creates audio-only channels from your music library  
**Best for:** Background music or specific genre channels  
**Example:** "Chill Lounge" or "Classic Rock Radio"

**How to create:**
1. Select **Type 12: Music**
2. Choose genre or create smart playlist
3. Music will play with visualizations
4. Configure duration and shuffle options

## Channel Configuration

### Basic Settings
- **Channel Name** - What appears in the guide
- **Channel Icon** - Visual identifier
- **Playback Mode** - Random, Sequential, or Resume
- **Episode Limit** - Maximum per channel rebuild
- **Reset Every X Hours** - Automatic refresh interval

### Advanced Options
- **Interleaving** - Mix in content from other channels
- **Bumpers** - Add channel identification videos
- **Commercials** - Insert ads or trailers
- **Resume Mode** - Control how playback continues
- **Excluded Content** - Block specific shows/movies

The organizer creates this professional layout:

**Channels 2-49: TV Shows**
- Channels 2-29: Non-documentary shows (alphabetically)
- Channels 30-49: Documentary & performance channels

**Channels 50-69: Movies**

**Channels 70-79: Music Videos**
- Dedicated to music video content

**Channels 80-89: Reserved**
- Space for future expansion

**Channels 90-99: Music**
- Audio-only music channels by genre

**Special Channels:**
- Channel 1: Never touched (your default)
- Holiday channels: Ordered by season

# Chapter 5: 10-Phase Preset System

## Understanding Presets

The 10-Phase Preset System allows you to create sophisticated daily viewing schedules. Each preset can control up to 10 different phases throughout your day, automating maintenance, startups, shutdowns, and channel changes.

## Available Presets

Paragon TV includes 5 customizable presets:
- **Alpha** - Typically for weekdays
- **Omega** - Weekend configuration
- **Delta** - Special occasions
- **Epsilon** - Guest mode
- **Gamma** - Vacation settings

## The 10 Phases Explained

### Phase 1: Maintenance
- Runs cleanup tasks
- Updates library
- Refreshes channel lineups
- **Typical time:** 4:00 AM

### Phase 2: First Startup
- Launches Paragon TV
- Tunes to morning channel
- Sets volume level
- **Typical time:** 6:00 AM

### Phase 3: First Shutdown
- Closes Paragon TV
- Optional blackout mode
- **Typical time:** 8:30 AM (after morning shows)

### Phase 4: Second Startup
- Midday activation
- Different channel/volume
- **Typical time:** 12:00 PM

### Phase 5: Second Shutdown
- Afternoon closure
- **Typical time:** 2:00 PM

### Phase 6: Third Startup
- Evening viewing begins
- Prime time settings
- **Typical time:** 5:00 PM

### Phase 7: Third Shutdown
- Brief break
- **Typical time:** 6:30 PM

### Phase 8: Fourth Startup
- Late evening settings
- **Typical time:** 7:00 PM

### Phase 9: Fourth Shutdown
- Pre-sleep shutdown
- **Typical time:** 11:00 PM

### Phase 10: Final Wake
- Late night option
- **Typical time:** 11:30 PM

## Configuring Presets

### Basic Setup
1. Go to **Settings** ‚Üí **Schedulers** ‚Üí **Preset System**
2. Select a preset (Alpha, Omega, etc.)
3. For each phase you want to use:
   - Set the time
   - Choose action type
   - Configure options

### Phase Options

**Maintenance Phases:**
- Select individual maintenance tasks
- Set force channel reset
- Configure library updates

**Startup Phases:**
- Enable/disable channel tuning
- Select starting channel
- Set volume level (0-100%)

**Shutdown Phases:**
- Choose exit type (Normal/Blackout)
- Configure warning time
- Set cleanup options

### Weekly Assignment

Assign presets to days:
1. Go to **Preset Schedule**
2. For each day, select a preset:
   - Monday: Alpha
   - Tuesday: Alpha
   - Wednesday: Delta
   - Thursday: Alpha
   - Friday: Omega
   - Saturday: Omega
   - Sunday: Epsilon

# Chapter 7: Speed Dial Navigation

## What is Speed Dial?

Speed Dial revolutionizes channel navigation by mapping your number keys (1-9) to instant actions. Instead of entering full channel numbers, single key presses jump to favorite channels or open show browsers.

## Two Speed Dial Modes

### 1. Channel Speed Dial
Map number keys to jump directly to channels:
- Press "1" ‚Üí Jump to Channel 15 (Sports)
- Press "2" ‚Üí Jump to Channel 12 (Comedy)
- Press "7" ‚Üí Jump to Channel 52 (Action Movies)

### 2. Show Speed Dial
Map directional combinations to TV show browsers:
- Press "Up + 1" ‚Üí The Office episode browser
- Press "Down + 2" ‚Üí Game of Thrones browser
- Press "Right + 3" ‚Üí Breaking Bad browser

## Configuring Speed Dial

### Method 1: Settings Menu
1. Go to **Settings** ‚Üí **Speed Dial**
2. For each slot (1-9):
   - Choose type: Channel or Show
   - Select target channel/show
   - Save settings

### Method 2: Long Press (Direct Mode)
1. Enable **Direct Mode** in settings
2. While watching, long-press any number
3. Menu appears:
   - Assign Channel
   - Assign Show
   - Clear Assignment

### Advanced Features

**Smart Detection:**
- Recognizes multi-digit entries
- 1-second timeout for combinations
- Visual feedback on screen

**Override Behavior:**
- Speed Dial takes priority
- Enter "05" for Channel 5 if "5" is mapped
- Hold "0" to temporarily disable

# Chapter 11: Multi-Room Synchronization

## Overview

Multi-Room Sync ensures all TVs in your home share the same Paragon TV experience. Change a channel in the living room, it updates everywhere.

## Sync Features

- **Channel Configuration** - All rooms share channel setup
- **EPG Data** - Same guide information everywhere
- **Playback Position** - Resume from any room
- **Favorite Shows** - Shared favorite lists
- **Settings Sync** - Most settings replicate

## Network Setup

### Requirements
- All Kodi instances on same network
- Static IP addresses (recommended)
- SSH enabled (LibreELEC)
- Shared storage or sync method

### Master/Client Configuration

**1. Designate Master System**
- Usually the main/living room system
- Most powerful hardware
- Always on

**2. Configure Clients**
Each additional room:
1. **Settings** ‚Üí **Multi-Room**
2. Set as **Client**
3. Enter Master IP address
4. Configure sync options

### Sync Options

**Real-Time Sync:**
- Channel changes propagate instantly
- Higher network usage
- Best for frequently used rooms

**Scheduled Sync:**
- Updates every X minutes
- Lower network impact
- Good for guest rooms

**Manual Sync:**
- Push updates on demand
- Minimal automation
- For rarely used systems

## File Sync Methods

### Method 1: Network Share
```
Master: /storage/.kodi/userdata/addon_data/script.paragontv/settings2.xml
Clients: Mount and symlink to same file
```

### Method 2: Rsync (Recommended)
```bash
# Client cron job (every 5 minutes)
*/5 * * * * rsync -av root@master:/storage/.kodi/userdata/addon_data/script.paragontv/settings2.xml /storage/.kodi/userdata/addon_data/script.paragontv/
```

### Method 3: Built-in Sync
- Paragon TV handles sync automatically
- Uses SSH for file transfer
- Configure in Multi-Room settings

# Chapter 12: Extended Filename Format

## Overview

The Extended Filename Format allows Paragon TV to extract rich metadata directly from filenames, eliminating the need for separate NFO files or online lookups.

## Format Structure

```
SSxEE - Episode Title - Show Title - Genre - Resolution - Audio Channels - Audio Codec - Holiday.ext
```

### Examples
```
01x01 - Pilot - The Office - Comedy - 1080 - 2 - AAC.mkv
02x05 - Halloween - Brooklyn Nine-Nine - Comedy - 1080 - 6 - DTS - Halloween.mkv
00x00 - A Charlie Brown Christmas - Peanuts - Animation - 720 - 2 - AAC - Christmas.mkv
```

## Components Explained

1. **SSxEE** - Season x Episode (00x00 for specials)
2. **Episode Title** - Name of the episode
3. **Show Title** - Series name
4. **Genre** - Primary genre
5. **Resolution** - 480/720/1080/2160
6. **Audio Channels** - 2/6/8
7. **Audio Codec** - AAC/DTS/DTS-HD/TrueHD
8. **Holiday** - Christmas/Halloween/Thanksgiving

## Using with Paragon TV

### Automatic Detection
Paragon TV automatically:
- Parses extended filenames
- Extracts all metadata
- Displays in EPG and overlays
- Uses for smart filtering

### Benefits
- **Faster Loading** - No NFO parsing needed
- **Richer Display** - More info in guides
- **Holiday Detection** - Automatic seasonal channels
- **Audio/Video Flags** - Visual quality indicators

# Chapter 15: Advanced Features & Global Rules

## Global Rules System

Global Rules apply content filters across ALL channels simultaneously, ensuring consistent viewing preferences.

### Available Rules

1. **Force Resume Mode**
   - Makes all channels resume from last position
   - Overrides individual channel settings

2. **Force Random**
   - All channels play content randomly
   - Disables sequential playback

3. **Force Real-Time**
   - Channels follow actual broadcast schedule
   - Content plays at specific times only

4. **Force New Episodes Only**
   - Only unwatched episodes appear
   - Great for catch-up viewing

5. **Force Lower Rating**
   - Excludes content above set rating
   - Family-friendly filtering

6. **Force Unwatched Only**
   - Hides all watched content
   - Perfect for discovery

7. **Exclude Shows Without Artwork**
   - Hides content missing posters
   - Cleaner visual experience

8. **Pause Commercials**
   - Auto-pause when commercials start
   - Manual resume required

9. **Exclude Specific Episodes**
   - Hide by episode number patterns
   - Useful for recap episodes

### Exclusion Management

**Global Exclusions:**
1. **Settings** ‚Üí **Advanced** ‚Üí **Global Exclusions**
2. Add shows/movies to exclude everywhere
3. Supports wildcards and regex

**Holiday Exclusions:**
- Hide Christmas episodes in July
- Control seasonal content display
- Date-based automatic filtering

## Channel Interleaving

Mix content from multiple channels:
- Channel 7 = 70% Drama + 30% Comedy
- Creates variety channels
- Maintains source channel rules

## Advanced Scheduling

**Strmode Options:**
- Resume - Continue from last position
- Random - Shuffle all content
- Sequential - Play in order
- Real-time - Follow schedule strictly

**Reset Modes:**
- Time-based - Reset every X hours
- Watch-based - Reset after X plays
- Manual only - User controlled
- Never - Persistent playlist

# Chapter 16: Troubleshooting

## Common Issues & Solutions

### Channels Not Playing

**Symptom:** Black screen or immediate skip

**Solutions:**
1. Check if content exists in Kodi library
2. Verify file paths are accessible
3. Clear channel and rebuild
4. Check codec compatibility
5. Review log for errors

### EPG Not Updating

**Symptom:** Old or missing guide data

**Solutions:**
1. Force EPG refresh (Settings ‚Üí Actions)
2. Check available storage space
3. Verify time/date settings
4. Clear EPG database
5. Increase EPG build time

### Slow Performance

**Symptom:** Laggy interface, slow channel changes

**Solutions:**
1. Reduce channel count
2. Lower EPG days (3 instead of 7)
3. Disable unused features
4. Clear thumbnail cache
5. Check CPU/memory usage

### Multi-Room Sync Issues

**Symptom:** Changes don't propagate

**Solutions:**
1. Verify network connectivity
2. Check SSH credentials
3. Confirm file permissions
4. Test manual sync
5. Review firewall settings

### Weather Not Showing

**Symptom:** Blank weather overlay

**Solutions:**
1. Configure Kodi weather service
2. Check internet connection
3. Verify API key (if required)
4. Update weather addon
5. Check overlay settings

## Debug Mode

Enable comprehensive logging:
1. **Settings** ‚Üí **Advanced** ‚Üí **Debug Mode**
2. Reproduce the issue
3. Check log at: `/userdata/temp/kodi.log`
4. Filter for: `[script.paragontv]`

## Getting Help
